Карточки

Карточки состоят из нескольких элементов, которые вы можете смешивать и сочетать:

  • .card: основной контейнер
    • .card-header : шапка карточки
      • .card-header-title : прибитый к левой стороне заголовок
      • .card-header-icon : место для иконки
    • .card-content : содержит контент карточки
    • .card-image : адаптивная картинка на всю ширину карточки
    • .card-footer : футер карточки, так же может содержать кнопки
      • .card-footer-item : стилизованные под карточку кнопки

Карточка адаптивна и подстроится под ширину родительского контейнера

Карточка с футером

Для того чтобы футер карточки отображался корректно, необходимо добавить класс .survey-card к классу .card Тогда футер прибьется к низу независимо от высоты карточки и займет 52px высоты

Уменьшенная карточка

Уменьшенная карточка используется для разбавления лейаута. По сути это та же самая карточка, но без картинки и с малым количеством текста.

Чтобы активировать стили уменьшенной карточки, добавьте к классу .card класс .type-2
