Элемены форм

Классические элементы форм, разработанные как для стандартных форма, так и для опросов

Поддерживаются все стандартные типы форм из нативных HTML-элементов:

  • <form>
  • <button>
  • <input>
  • <textarea>
  • <label>

Для указанных типов форм форм поддерживаются следующие классы:

  • .label
  • .input
  • .textarea
  • .select
  • .checkbox
  • .radio
  • .button
  • .help

Для равномерности распределения элементов и их дизайна есть полезный контейнер .control, в который можно оборачивать элементы управления формой.

При объединении нескольких элементов управления в форме, используйте класс .field в качестве контейнера. Таким образов сохраняется согласованность отступов.
