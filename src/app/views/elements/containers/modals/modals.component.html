Модальные окна

Классическое модальное окно с оверлеем. Адаптивно и можно использовать любой тип контента

Модальные окна делятся на 4 типа:

  • Стандартное - к контейнеру .mat-dialog-box не добавляется никаких классов. Модалка занимает столько места, сколько занимае контент. Адативно для всех разрешений
  • .is-normal - к контейнеру .mat-dialog-box добавляется класс .is-normal. Окно ограничивается по ширине в 400px, адаптивно для всех разрешений.
  • .is-medium - к контейнеру .mat-dialog-box добавляется класс .is-medium. Окно ограничивается по ширине в 600px, адаптивно для всех разрешений.
  • .is-fluid - к контейнеру .mat-dialog-box добавляется класс .is-fluid. Окно не ограничивается по ширине, старается занять как можно больше пространства. Адаптивно для всех разрешений.
  • .is-big - к контейнеру .mat-dialog-box добавляется класс .is-big. Окно ограничивается по ширине в 768px, адаптивно для всех разрешений.
