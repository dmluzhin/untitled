Кнопки

Классические кнопки, разных цветов, размеров и состояний
Типы кнопок

Существует три типа кнопок:

  • <button> - стандартная кнопка
  • <a> - анкорная ссылка
  • <input type="submit"> - инпут для форм и прочего

Все типы кнопок выглядят одинаково при наличии класса .button

Цвета

Кнопки доступны в разных цветах. Цвета определяются переменными в файлe variables.scss

Чтобы изменить цвет стандартной кнопки необходимо добавить класс-модификатор, например класс .is-primary добавит кнопке брендовый цвет
Размеры

Кнопки могут быть в четырех разных размерах:

  • Крошечная
  • Маленькая
  • Стандартная
  • Средняя
  • Большая

Чтобы изменить размер кнопки, необходимо добавить один из четырех модификаторов:

  • .is-tiny - делает кнопку крошечного размера
  • .is-small - делает кнопку маленького размера
  • .is-medium - делает кнопку среднего размра
  • .is-large - делает кнопку большого ращмера

Чтобы оставить кнопку стандартного размера, просто не добавляйте никаких классов

Размеры кнопок справедливы для всех типов кнопок.

Вы так же можете применить нужный размер сразу для нескольких кнопок одновременно. Для этого добавьте один из приведенных ниже классов к контейнеру <button class="buttons">:

  • .are-small - делает кнопки маленького размера
  • .are-medium - делает кнопки среднего размера
  • .are-large - делает кнопки больгого размера

Однако, если это необходимо, вы можете изменить размер одной или несколких кнопок из набора добавить один из уже классов к самой кнопке:

  • .is-tiny - делает кнопку крошечного размера
  • .is-small - делает кнопку маленького размера
  • .is-medium - делает кнопку среднего размра
  • .is-large - делает кнопку большого ращмера
Резиновые кнопки

Кнопку любого размера можно сделать полноразмерной, или резиновой. Чтобы использовать полноразмерную кнопку, нужно добавить к кнопке класс .is-fullwidth. Кнопка займет все доступное пространство родительского блока и будет тянуться в зависимости от его размера.

Тумблеры

Тумблеры - те же кнопки, но могут иметь два состояния отключена (блекло синий) и вкючена (зеленый) по принципу чекбокса
